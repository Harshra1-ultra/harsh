Theoretical Probabilities

Amplitude Probability Density Function (PDF)
\[f_A(x) = \frac{x}{\sigma^2} \exp\left(-\frac{x^2+s^2}{2\sigma^2}\right) I_0\left(\frac{x s}{\sigma^2}\right), \quad x \ge 0\]
Phase Probability Density Function (PDF)
\[f_{\Phi}(\phi) = \frac{1}{2\pi} e^{-\frac{s^2}{2\sigma^2}} + \frac{m_{\text{til}}}{\sqrt{2\pi}\sigma} e^{\frac{m_{\text{til}}^2 - s^2}{2\sigma^2}} Q\left(-\frac{m_{\text{til}}}{\sigma}\right)\]
where \(m_{\text{til}} = m_1\cos\phi + m_2\sin\phi\) and \(s = \sqrt{m_1^2 + m_2^2}\).
